CoreMedia Content Management Server Nagios Plugin
=================================================

Nagios Plugin - check_coremedia

Prerequisites
-------------
- Monitoring::Plugin
- CoreMedia CMS Tools

### RHEL/CentOS
`sudo sudo yum install perl-Monitoring-Plugin`

Usage
-----
```
./check_coremedia --component cms --prefix=cm8 --path=/opt/coremedia
```
